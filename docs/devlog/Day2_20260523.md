**Day 2**
- Tasks: OpenDART MCP server / K-IFRS MCP server / K-IFRS chunking build script
- Details: MCP 서버를 구축하여 OpenDART API를 호출하고 적합한 데이터를 파싱하는 10개의 도구 스펙을 작성했고, K-IFRS 기준서가 임베딩된 DB에서 쿼리에 대한 답변을 도출하기 위한 의미 기반 검색 툴과 기준서 번호, 문단을 직접 질문할 경우를 고려해 직접 조회 툴을 추가했다. 이후 K-IFRS 기준서를 임베딩하기 전 우선 문단별로 청킹하고 너무 긴 문단의 경우에는 서브청크로 다시 나누었다. 이 과정에서 문단 번호를 매치하기 위한 정규식 문법을 새로 학습하였다. 
- Learn: 정규식 문법 (re)
         re.compile(...): 정규식을 미리 컴파일 해둠
         r"...": 백슬래시를 글자 그대로 취급하는 raw string
         ^: 줄의 시작
         \s*: 공백 0개 이상 (들여쓰기 허용)
         (...): 캡처 그룹. 매칭된 내용을 m.group(1)로 꺼낼 수 있다
         (?:...): 비캡처 그룹. 추출 불가능
         |: or
         \d: 0-9 숫자
         \.: 마침표
         .: 임의의 문자
         ?: 앞 요소가 0~1번
         [가-힣]: 한글 한 글자
         \s: 공백 1개 (이후에 본문 시작)
