- Tasks: Streamlit_app.py 구현 및 테스트 / Citation 데이터타입 수정 / 인용 추출 함수 추가

- Details: 
    프론트엔드를 간단히라도 구현해보기 위해 Streamlit을 공부하고 활용해봄.

    업무상 활용을 위해 가장 중요한 것은 명확한 출처라고 생각하여 Citation 데이터타입을 더 세분화 및 구체화하고 에이전트 호출별로 케이스를 나누었다. 이를 streamlit에도 적용하여 답변 시 최대한의 구체적 근거를 제공할 수 있게끔 함.

    구현한 Streamlit UI에서 직접 테스트를 진행해보았는데 AI가 이전 답변의 맥락을 파악하지 못하는 문제가 발생하여 대화 기록 전달이 누락되었다는 사실 인지하고 수정함.

- Learn:
    Streamlit: 
    파이썬 코드로 데이터 웹 애플리케이션을 구축할 수 있는 오픈소스 라이브러리. HTML, CSS, JS를 몰라도 UI컴포넌트가 모두 파이썬 함수로 제공됨. 프로토타이핑이 빠르고, Matplotlib, Seaborn 등 파이썬 시각화 라이브러리와도 완벽히 호환됨. 

    사용자가 위젯(버튼 클릭, 텍스트 입력)과 상호작용을 하면 파이썬 스크립트 전체를 위에서부터 아래로 처음부터 끝까지 다시 실행함. 이는 복잡한 상태 변경 이벤트 처리를 단순화해 주지만 데이터 로딩이나 무거운 머신러닝 모델 추론 코드가 있을 때 웹 페이지가 매우 느려질 수 있음. 이 때문에 Caching 기능을 제공함. 
    @st.cache_data: 데이터 프레임 로드, API 호출, 파일 읽기 등 데이터 자체 저장 시 사용.
    @st.cache_resource: 머신러닝 모델 객체, 데이터베이스 커넥션 등 한 번 생성 후 계속 재사용해야 하는 무거운 객체를 저장할 떄 사용.
    
    1. 텍스트 및 미디어 출력
        st.title(), st.header(), st.subheader(): 제목 및 소제목 정의
        st.write(): 텍스트, 데이터프레임, 차트, 에러 메시지 등 객체의 종류에 따라 알맞은 형태로 자동 렌더링
        st.markdown(): 마크다운

    2. 데이터 디스플레이
        st.dataframe(): 인터렉티브 표(정렬, 검색 가능) 출력
        st.table(): 정적인 고정 표 출력
        st.json(): JSON 구조를 깔끔하게 접고 펼 수 있는 형태로 출력

    3. 인터랙티브 위젯
        st.button(): 버튼
        st.text_input() / st.text_area(): 텍스트 입력 창
        st.slider(): 범위나 숫자를 조절하는 슬라이더
        st.selectbox() / st.multiselect(): 드롭다운 선택 상자
        st.file_uploader(): 사용자가 파일(CSV, 이미지 등)을 업로드

    4. 레이아웃 및 내비게이션
        st.sidebar: 좌측에 고정된 사이드바 메뉴 생성
        st.columns(): 화면을 가로로 분할하여 다단 레이아웃 구성
        st.tabs(): 탭 인터페이스 구현
