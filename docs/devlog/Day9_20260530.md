- Tasks: 최종 테스트 및 기능 점검

- Details: 
    인용 목록을 표시하는 streamlit의 expander가 나타나지 않아 _extract_citations 함수에 테스트 코드를 추가하여 ToolMessage가 제대로 전달되는지 확인함. search_kifrs ToolMessage가 없는 것을 확인하여 worker 내부 메시지를 노출하게끔 output_mode를 "full_history"로 변경함.
    그러자 payload .get AttributeError가 발생. langchain_mcp_adapters가 MCP 표준의 TextContent 래핑 ([{"type": "text", "text": "<json>}])을 그대로 ToolMessage.content에 담아 전달하기 때문에 한 단계 더 파싱해서 내부 dict를 꺼내는 _parse_tool_payload 헬퍼를 추가함.
    full_history로 컨텍스트를 전부 누적시키자 토큰이 소모되는 속도가 매우 빨라진 것을 체감. 우선은 기능 최대화를 위해 유지하고 추후 상황에 맞게 수정하는게 맞다고 판단함.
